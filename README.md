## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/DristiMeet.git

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Start backend
cd ../backend
npm run dev

# Start frontend
cd ../frontend
npm start
```
